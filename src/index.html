String va Number

Nega"6" + 3 = “63” lekin “6” - 3 = 3
  • JavaScriptda + operatori raqamlar bilan ishlaganda qo'shadi, lekin agar operandlardan biri yoki ikkalasi ham string (matn) bo'lsa, + operatori ularni yonma-yon ravishda bir-biriga qo'shib qo'yadi (bu jarayon "string concatenation" deb ataladi).
  • Farqli ravishda, -, *, / operatorlari faqat sonlar bilan ishlaydi. Agar operandlar string bo'lsa ham, JavaScript avtomatik ravishda ularni raqamlarga aylantiradi va keyin arifmetik amal bajaradi.
  • Agar matematik operatsiya yoki hisoblash jarayonida noto'g'ri yoki mantiqsiz natija chiqsa, JavaScript NaN qiymatini qaytaradi.

undefined (aniqlanmagan qiymat)
